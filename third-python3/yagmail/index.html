Yagmail-邮件发送包

2019年11月
编程设计, 其它
展开书签
  • [1] 包所在地址
https://github.com/kootenpv/yagmail
  • [2] 包安装
pip3 install yagmail
  • [3] 包用法
#!/bin/python3

# 邮件正文，html中嵌入sound,vedio images file
contents = ['This is the body, and here is just text http://somedomain/image.png',
            'You can find an audio file attached.', '/local/path/song.mp3']
# smtp starttls认证
yag = yagmail.SMTP(user="aaa@163.com",password="123",host="smtp.163.com",port=587,smtp_starttls=True, smtp_ssl=False)

# smtp plain认证
yag = yagmail.SMTP(user="aaa@163.com",password="123",host="smtp.163.com",port=25,smtp_starttls=False, smtp_ssl=False)

status = yag.send(to="bbb@163.com",title="邮件标题",contents)
print(status)
  • [3] send方法使用
to = 'santa@someone.com'
to2 = 'easterbunny@someone.com'
to3 = 'sky@pip-package.com'
subject = 'This is obviously the subject'
body = 'This is obviously the body'
html = '<a href="https://pypi.python.org/pypi/sky/">Click me!</a>'
img = '/local/file/bunny.png'
  • send()的方法定义：
def send(self, to=None, subject=None, contents=None, attachments=None, cc=None, bcc=None,preview_only=False, validate_email=True, throw_invalid_exception=False, headers=None)

＃ 不带附件，仅内容
yag.send(to = to, subject = subject, contents = body)
＃ 带附件
yag.send(to = to, subject = subject, contents = [body, html, img])、
＃ 多接收用户，用元表
yag.send(to=[to1,to2,to3],subject = subject, contents = [body, html, img])
# 匿名
yag.send(contents = [body, img])
# 多附件
yag.send(to=[to1,to2,to3],subject = subject, contents = [body, html, img],attachments=['./tmp.txt','./tmp2.txt])
  • 详细可以详阅官方文献：https://github.com/kootenpv/yagmail
