HowTos:Asterisk Call Parking 呼叫停泊功能配置

配置features.conf

blindxfer=##  ;转接开始的前缀
atxfer=*2

电话转接功能的时间间隔调节：

1.转接代码的按钮时间间隔(##或*2)
修改 features.conf，如果是FreePbx则修改features_general_custom.conf。
; Max time (ms) between digits for
; feature activation. Default is 500
featuredigittimeout = 1500 

2.启动指定转接后等待拨号时间间隔：
修改 features.conf，如果是FreePbx则修改features_general_custom.conf。
; Number of seconds to wait between digits when transferring a call
; (default is 3 seconds)
transferdigittimeout => 8 ;你按##后，到你拨第一个号码按键的时间。

转接后将来电号码带过去

sip.conf
    sendrpid=pai
    trustrpid=yes

extensions.conf
	exten => _x.,1,Dial(DAHDI/G0/${EXTEN},30,t) ;注意加了t参数，t的意思是允许被叫用户按前面设置的##转接；相应的T是允许主叫用户按##转接。
	exten => _8XX,1,Dial(SIP/${EXTEN},60,oM(recording,${CALLERID(num),${EXTEN}}),tT);

Feature Codes

Asterisk 内建了许多很好用的电话操作功能， 这些统称为 Feature Codes(功能代码)， 以下将列出比较常用的功能操作码。

《来电转接》	 
启动来电转接	＊72
关闭来电转接	＊73
来电转接语音关闭	＊74
启动忙线转接	＊90
关闭忙线转接	＊91
忙线转接语音关闭	＊92
启动未接来电转接	＊52
关闭未接来电转接	＊53

《系统服务》	 
Echo Test	＊43
查询你的分机号码	＊65
语音报时	＊60

《语音信箱》	 
听取目前分机的语音信箱	＊97
听取其他分机的语音信箱	＊98
给分机留言	＊<分机号>

《其他》	 
 电话唤醒(Wakeup/Morning Call)	＊62
 通话转接(Blind Transfer) 	＃＃<目的分机号> 
 通话指定转接(Attended Transfer)	＊2 
 三方通话(注：这是终端电话的功能)	用户 A 与用户 B 通话中，用户 A 按 Flash/Hook 键保留 B 的通话，用户 A 话筒出现 Tone 音，接着拨用户 C 分机，用户 A 再按 
Flash/Hook 键，完成三方通话。

来电代接	＊＊<分机号>
来电群组代接	＊8
通话录音	＊1
