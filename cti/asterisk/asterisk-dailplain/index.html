asterisk extension.conf

拨号计划语法

[default]  ;;这个是卡片context
exten =>  ;;;这为执行命令,
exten => name,priority,application()

  • 分机的名字（或号码）

  • 优先级（分机包含很多步骤；步骤的序号被称为“优先级”）,连续序号可用n动态表示

  • 应用程序（或者说命令），该步骤要执行的动作,详细看Dial() Application

示例： 优先级 不编号的优先级 “same =>”同执行命令下操作符 优先级标号

exten => s,1,NoOp(此处最先级显示注释内容)
exten => s,n,Answer()  ;;n 不编号的优先级
	same => n(end),Hangup() ;;(end) 对n不编号优先级特别标识，作用用于跳转，如Goto function

处理无效入口和超时

如果一个 context 收到了 一个针对不存在的 extension 的请求(例如,在我们上面的例子中输入 9),呼叫会被转给 i extension 处理。我们还需要一个 extension 来处理当用户在给定的时间(默认的超时时间是 10 秒)内没有按下任何按键的情况。如果用户在 WaitExten()被调用后太长时间没有按下按 键,呼叫会被传递给 t extension。下面是增加了这两个 extension 后的 dialplan:

  [TestMenu]
    exten => start,1,Answer()
         same => n, Background(main-menu)
         same => n, WaitExten(5)
 
  exten => 1,1,Playback(digits/1)
     same => n, Goto(TestMenu,start,1)

  exten => 2,1,Playback(digits/2)
     same => n, Goto(TestMenu,start,1)

  exten => i,1,Playback(pbx-invalid)
     same => n, Goto(TestMenu,start,1)

  exten => t,1,Playback(vm-goodbye)
     same => n, Hangup()

extension 通配符

X和x表示单个0-9的数字
N和n表示单个1-9的数字
Z和z表示单个2-9的数字
.表示单个的任何字符和数字
[]比如你想匹配1、3或者6，[1,3,6]，[1-8]匹配1到8的任意一个数字,详细请看正则表达式基本部分
s ：Start extension，也就是当没有extension的时候就会执行这个流程（例如在模拟外线进线没有收到callerID的情况下就会转到这个extension来执行）
t ：timeout extension，也就是说如果等待用户输入超时后就会转到t这个流程来执行，在这里你可以设置一些提示音来告诉客户超时了。
i ：invalid extension，也就是说如果客户输入无效的时候会转到i这个流程来执行
fax ：fax calls，也就是说如果asterisk检测到传真信号的时候就会自动转到这个流程里面来执行。
h ：hangup extension，就是说呼叫终止后执行的流程，在这里通道已经终止，放音、发送DTMF等命令都不可用了，只能做一些呼叫结束后处理的一些工作。

include语句中可以加入对时间限制的功能，其格式如下：
include => context | <time> | <day> | <day-of-month> | <month>
其中day和month使用其前3个英文字母表示，如：
星期: mon,tue,wed,thu,fri,sat,sun
月：jan,feb,mar,apr...
时间使用24小时格式。
例子：
A business is open from 9:00 a.m. until 5:00 p.m. Monday to Friday and

from 9:00 a.m. to 2:00 p.m. Saturday. The dialplan would look like this:

; Day
include => open|09:00-17:00|mon-fri|*|*
include => open|09:00-14:00|sat|*|*
include => closed

[open]
exten => 2000,1,Dial(SIP/2000)

[closed]
exten => 2000,1,VoiceMail(2000,u)

nano extension.conf

  [default]
      include => dialout
  [dialout] ;本地呼出
    exten => _XXXXXXX.,1,Dial(SIP/${CALLERID(num):0:3}/${EXTEN},60,moM(recording,${CALLERID(num)},${EXTEN}));
    ;本地内线如810000呼叫13800138000
    ;_XXXXXX. 表示为最小六位号码
    ;${CALLERID(num):0:3 表示截取前三位号为810,
    ;${EXTEN} 表示为目标号码
    ;Dial()采用810分机呼叫13800138000
    
    [to-company]
    	exten => _XXX,1,Dial(SIP/${EXTEN}000@localhost,60,moM(recording,${CALLERID(num)},${EXTEN}));
    	;入局到XXX(810) 号码转呼XXX000(810000) 接着调用宏[marco-recording]进行录音

    [marco-autodialout] ;出局线路自动轮转呼出宏
	exten => s,1,NoOp()
	exten => s,n(dial2),Dial(DAHDI/4/${ARG2},60,M(recording,${ARG1},${ARG2})); 线路4
		same => n,GotoIf($["${DIALSTATUS}" = "ANSWER"]?end);应答，否转入下条
		same => n(dial3),Dial(DAHDI/g0/968200${ARG2},60,M(recording,${ARG1},${ARG2})); 线路g0跑200电话
		same => n,GotoIf($["${DIALSTATUS}" = "ANSWER"]?end);应答，否转入下条
		same => n(dial4),Dial(DAHDI/g0/${ARG2},60,moM(recording,${ARG1},${ARG2})); 线路跑200失败转直线跑
		same => n(end),Hangup();

    [macro-autodialoutswitch]
	exten => s,1,NoOp()
	exten => s,n,Dial(SIP/${ARG1:0:3}/${ARG2},60,moM(recording,${ARG1},${ARG2}));
        	same => n,GotoIf($["${DIALSTATUS}" = "ANSWER"]?end)
        	same => n,Dial(SIP/888888/${ARG2},60,moM(recording,${ARG1},${ARG2}))
        	same => n(end),Hangup();

   
    [marco-recording] ;录音宏
	exten => s,1,Set(RECORDDIR1=/usr/local/asterisk/monitor);
	exten => s,n,Set(RECORDDIR2=${STRFTIME(${EPOCH},,%Y)}/${STRFTIME(${EPOCH},,%m)}/${STRFTIME(${EPOCH},,%d)});
	exten => s,n,Set(RECORDFILENAME=${STRFTIME(${EPOCH},,%H%M%S)}-${ARG1}-${ARG2}.WAV);
	;exten => s,n,Set(RECORDFILENAME=${STRFTIME(${EPOCH},,%H%M%S)}-${UNIQUEID}.wav);wav为超高压缩率可减少线路开销，但网页播放插件仅苹果支持。2016.4.19
	exten => s,n,Set(RECORDPATH=${RECORDDIR1}/${RECORDDIR2}/${RECORDFILENAME});
	exten => s,n,Set(CDR(userfield)=${RECORDDIR2}/${RECORDFILENAME}); 此处字段【userfield】asterisk 13 后支持自定义字段，可以自设
	exten => s,n,MixMonitor(${RECORDPATH});


    [incoming-mobile]
        exten => sms,1,Verbose(Incoming SMS from ${SMSSRC} ${SMSTXT})

        exten => _[8,9]XXXXX,n,MessageSend(SIP:${EXTEN}@localhost",${CALLERID(name)}"${MESSAGE(from)})

1000,2,Set(DYNAMIC_FEATURES=testfeature#musichold#record#unrecord)

exten => i,1,playback(pbx-invalid)
exten => i,2,goto(default,1000,1)
exten => t,1,playback(vm-goodbye)
exten => t,2,hangup()
exten => 1,1,goto(internal,s,1)
exten => _9.,1,goto(out-bound,s,1)  //拨9呼外线，需另外写out-bound context。
exten =>
500,1,voicemailmain()      //收听语音信箱内容
exten => 500,2,hangup()
